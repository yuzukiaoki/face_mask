專案製作此程式構思為，當前疫情肆虐，各地方政府皆在加強宣導戴口罩，但光是漫無目的宣導不會有明顯的成效，針對特定區域去做錄影，再根據此區域錄影影片的統計資料輸出圖表，可以一目了然看出該區域
口罩配戴狀況，並決定加強宣導的與否。
